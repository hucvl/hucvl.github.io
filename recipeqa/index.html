Evaluation and Submission


Training dataset and a dev-set of RecipeQA Challenge are below.



To evaluate your models, we provide an evaluation script that will be used for the official evaluation, along with a sample prediction file. Evaluation script can be run as:
python evaluate.py {path-to-prediction-file} {path-to-dev-set path-to-output-file}



We used CodaLab platform to retrieve submisions of models of competitors. CodaLab also provides a good competition creation and handling mechanism so that we can test your models with private dataset easily.
